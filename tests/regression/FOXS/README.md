# FOXS Regression Tests

This directory contains regression tests for the SAXS featuriser, validating that
biofeaturisers' computed SAXS profiles match standard FoXS (Fast Oriented X-ray Scattering) 
tool output.

## Test Structure

### 5PTI Regression Test

The primary test uses the **5PTI NMR structure** (bovine pancreatic trypsin inhibitor):
- **PDB ID**: 5PTI
- **Frame used**: First model (frame 0)
- **Preprocessing**: Solvent atoms removed (HOH, WAT, ions), hydrogens kept, deuterium (D) normalized to hydrogen (H)
- **Global Availability**: The 5PTI structure is registered as a global pytest fixture (`pdb_5pti_frame0`) 
  in `tests/conftest.py` and is available to all tests across the suite.

#### Test Files

- **`fixtures/5PTI_frame0.pdb`**: Preprocessed structure (downloaded from PDB, solvent removed)
- **`fixtures/5PTI_frame0_foxs_reference.dat`**: Reference SAXS profile (generated by FoXS tool or biofeaturisers)
- **`test_FOXS_regression.py`**: Test that validates numerical agreement between biofeaturisers SAXS and reference
- **`conftest.py`**: FOXS-specific pytest fixtures (reference profile loader)
- **`generate_foxs_reference.sh`**: Script to regenerate reference profile using FoXS (requires `imp_env`)

#### Running Tests

```bash
# Run the FOXS regression test
pytest tests/regression/FOXS/test_FOXS_regression.py -v

# Run with verbose output to see profile statistics
pytest tests/regression/FOXS/test_FOXS_regression.py -vv
```

## Using the 5PTI Fixture in Other Tests

The 5PTI structure is available globally via the `pdb_5pti_frame0` fixture:

```python
# In any test file in the biofeaturisers project:

def test_your_feature(pdb_5pti_frame0):
    """Your test that uses the 5PTI structure."""
    from biotite.structure import AtomArray
    atom_array: AtomArray = pdb_5pti_frame0
    
    # Use atom_array for your test...

```

## Regenerating Reference Data

If you update the structure preprocessing or want to regenerate the FoXS reference profile:

### Prerequisites

Create the IMP environment with FoXS:

```bash
conda create -n imp_env -c salilab imp
```

### Regenerate Reference Profile

```bash
cd tests/regression/FOXS
./generate_foxs_reference.sh
```

This script:
1. Reads the preprocessed PDB file (`fixtures/5PTI_frame0.pdb`)
2. Runs the FoXS tool with standard parameters:
   - Include explicit hydrogens (`-h` flag)
   - Default: `profile_size=500`, `max_q=0.50`, `c1=1.0`, `c2=0.0`
3. Saves the resulting `.dat` file as `fixtures/5PTI_frame0_foxs_reference.dat`

## FoXS Parameters

The reference profile is generated with standard FoXS parameters:

- **Profile size**: 500 points
- **q range**: 0.01 to 0.50 Å⁻¹
- **Hydration layer**: Included (FoXS default)
- **Fitting parameters**: c1=1.0, c2=0.0 (no adjustments)
- **Element form factors**: Water Kirby (WK) with hydrogens explicit

See [FoXS documentation](https://imp.readthedocs.io/en/latest/) for parameter details.

## Test Validation

The test validates that biofeaturisers' computed profiles match the FoXS reference using:

- **Relative tolerance**: 5% (rtol=0.05)
- **Absolute tolerance**: 10% of mean intensity (atol=0.1 × mean(I))

This allows for minor variations due to:
- Different hydration layer implementations
- Numerical precision differences
- Form factor tabulation resolution

## Notes

- The 5PTI structure was chosen as a test case because it is well-characterized, publicly available, and 
  compact enough to compute quickly while representative of protein structures in SAXS experiments.
- The first frame is used from the NMR ensemble to provide a single, deterministic test case.
- Both hydration layers are included in the FoXS computation for realistic scattering profiles.
